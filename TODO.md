# 🛠️ Próximas Mejoras y Correcciones (Roadmap)

Este documento detalla los fallos técnicos y de UX identificados que serán resueltos en las próximas versiones para garantizar un producto de calidad profesional.

### 🔴 Prioridad Alta (Errores de Interfaz)
- [ ] **Corrección Ortográfica:** Revisar tildes en `Sesión`, `Ubicación`, `Categorías` y `Descripción`.
- [ ] **Fix Gramatical:** Corregir signos de exclamación en pantalla de éxito (actualmente `!Texto¡`).
- [ ] **Fix de Typos:** Corregir `Trasnferencia` por `Transferencia` en métodos de pago.

### 🟡 Prioridad Media (UX / Consistencia)
- [ ] **Unificación Monetaria:** Homologar el uso de Lempiras (L) en toda la app para evitar discrepancias con el signo `$`.
- [ ] **Validación de Inputs:** Implementar `showDatePicker` en el campo de fecha para evitar errores de formato manual.
- [ ] **Refactor de Labels:** Corregir el placeholder "Dale un nombre a tu venta" dentro del módulo de Gastos.

### 🟢 Prioridad Baja (Escalabilidad)
